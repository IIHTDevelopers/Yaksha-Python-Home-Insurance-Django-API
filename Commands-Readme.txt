* Install ‘djangorestframework’ module before running the code. For this use the following command. 

	pip install djangorestframework

* Use the following command to run the server
	
	python3 manage.py runserver
 
* Mandatory: Before final submission run the following commands to execute testcases
	
	python3 manage.py test insuranceapp.test.test_functional
   	python3 manage.py test insuranceapp.test.test_exceptional
   	python3 manage.py test insuranceapp.test.test_boundary
